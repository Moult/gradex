# usydarchexhibition

Website for the University of Sydney Architecture Exhibition

# Installation

 1. `curl -s https://getcomposer.org/installer | php`
 2. `php composer.phar install --dev`
 3. Configure `build.properties` and `behat.yml`
 4. `bin/phing -l`
